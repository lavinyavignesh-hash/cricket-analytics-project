# cricket-analytics-project

# 🏏 T20 Cricket Data Analysis (Batting & Bowling)

## 📌 Project Overview
This project focuses on analyzing T20 cricket data to understand player and team performance.  
Using Python and Power BI, the data is cleaned, analyzed, and visualized to generate meaningful insights for better decision-making.

---

## 🎯 Problem Statement
The objective of this project is to analyze T20 cricket data and derive insights into batting and bowling performance using key metrics such as runs, strike rate, wickets, and economy. This helps in identifying top performers and supports data-driven decisions in team strategy.

---

## 📊 This Project Focuses On
- Cleaning and preparing raw cricket data  
- Performing exploratory data analysis (EDA)  
- Identifying key performance factors  
- Comparing players and teams  
- Creating visualizations using Python  
- Building interactive dashboards in Power BI  
- Generating data-driven insights  

---

## 🗂️ Dataset Information
The dataset includes:

### 🏏 Batting Data
- Runs, Balls, Strike Rate  
- 4s, 6s  
- Player and Team details  

### 🎳 Bowling Data
- Overs, Runs conceded  
- Wickets, Economy rate  
- Wides and No-balls  

### 📊 Match Data
- Teams, Winners  
- Venue, Match details  

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Power BI  

---

## 📈 Key Insights

### 🏏 Batting
- Most players have strike rates between 80–140  
- Strong relationship between runs and balls faced  
- Boundaries significantly impact total runs  

### 🎳 Bowling
- Most bowlers maintain economy between 5–9  
- Positive relationship between overs and wickets  
- Some bowlers are highly efficient with low economy  

---

## 💡 Business Recommendations
- Select players with high strike rate and consistency  
- Focus on bowlers with low economy and good wicket-taking ability  
- Optimize team strategy using data-driven insights
